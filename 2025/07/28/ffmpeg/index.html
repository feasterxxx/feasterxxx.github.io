FFmpeg

  1. 安装
    ffmpeg官网
    下载版本https://www.gyan.dev/ffmpeg/builds/packages/ffmpeg-7.0.2-essentials_build.7z
    解压后会
    解压后的文件夹中应包含以下目录：
    bin：FFmpeg 可执行文件所在的文件夹，运行 FFmpeg 的所有命令都需通过此目录下的文件
    doc：文档资料
    presets：预设的格式和编码方案
    进入 bin 目录，可以看到 FFmpeg 的三个核心可执行文件：ffmpeg.exe、ffplay.exe、ffprobe.exe

  2. 配置
    在系统变量“PATH”中添加bin目录路径

  3. 测试
    在cmd窗口中输入命令ffmpeg -version
    得到以下输出表示配置成功

  4. 使用
    FFmpeg 命令行格式ffmpeg [global_options] {[input_file_options] -i input_url} ... {[output_file_options] output_url}

         参考链接：https://blog.csdn.net/Natsuago/article/details/143231558

  5. 音频转换命令

  • ncm转flac
    ffmpeg -i input.ncm -codec:a flac output.flac
    -i input.ncm : 指定输入文件，这里的 input.ncm 是要转换的NCM文件名。
    -codec:a flac : 使用FLAC编码器进行音频编码。
    output.flac : 指定输出文件名，将转换后的文件保存为FLAC格式。

  • flac转mp3
    ffmpeg -i input.flac output.mp3

  1. 批量转换 input.flac 是输入的 FLAC 文件名。
    output.mp3 是输出的 MP3 文件名。
    -ab 320k 设置比特率为 320kbps（可根据需求调整）。
    -map_metadata 0 保留元数据。
    -id3v2_version 3 确保 MP3 的元数据兼容性。
